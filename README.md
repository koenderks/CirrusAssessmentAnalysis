# Cirrus Assessment Analysis Application

A Shiny application for generating comprehensive assessment reports. This app allows educators and assessment designers to analyze test data, visualize distributions, evaluate item and test statistics, and produce a polished HTML report. The app is built with R, **Shiny**, **ggplot2**, and **kableExtra**, and is designed for clarity, professional presentation, and actionable insights.

---

## Features

- **Descriptive Statistics**: Compute and display key metrics such as number of participants, mean, median, standard deviation, and skewness.  
- **Histogram Visualization**: Visualize score distributions to detect ceiling/floor effects.  
- **Test-Level Statistics**: Evaluate assessment difficulty, discrimination (RIT/RIR), and reliability (Cronbach’s alpha).  
- **Item-Level Statistics**: Analyze individual items with P-values, item-total correlations, item-rest correlations, and impact on reliability if deleted.  
- **Item Plots**: Scatter plots showing item difficulty versus discrimination.  
- **Item Correlation Matrix**: Heatmap to identify relationships between items and potential redundancies.  
- **HTML Report Generation**: Export a fully formatted, professional HTML report with color-coded metrics, embedded plots, and annotations.
